We are pleased to present our own Web API

Web API provides a user with compact and quick list of character's data, taken and built from the universe of HxH anime.
In order to use, you don't need to get any key or token. Just name your character and let's go!

API response format:

You will get a response of JSON formatted data.
In fact, at this time it includes much more data than what is displayed on the website.

How should I use it?

Send a query request to the https://hellohunter.herokuapp.com/api/characters?name=, where the key name must have a value with the character's name.
GET/ requests supported only!

Examples:

GET/: https://hellohunter.herokuapp.com/api/characters?name=Killua
...
Status: 200 (OK), JSON response:
 {
  "id": 5,
  "name": "Killua Zoldyck",
  "age": "12",
  ...
 }

GET/: https://hellohunter.herokuapp.com/api/characters?name=IsaacNetero
...
Status: 200 (OK), JSON response:
 {
  "id": 11,
  "name": "Isaac Netero",
  "age": "110+",
  ...
 }

GET/: https://hellohunter.herokuapp.com/api/characters?name=Naruto
...
Status: 404 (Error), Not Found:
 {
  "userRequest": "Naruto",
  "result": "Sorry, no data found :("
 }
